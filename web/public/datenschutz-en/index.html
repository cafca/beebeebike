2. Overview

beebeebike is a bicycle-friendly route planner for Berlin. We process personal data solely to provide the service. There is no sale and no disclosure for advertising purposes.

Map tiles (MapLibre) and the routing engine (GraphHopper) are operated by us on a server rented from Hetzner Online GmbH (Gunzenhausen, Germany). There is no forwarding to external map providers (e.g. Google, Mapbox) or external routing services.

4. Guest mode

You can use beebeebike anonymously without an account. In that case only a random session ID is set as a cookie. Ratings you create as a guest are stored server-side bound to this session ID so they remain available on later visits. No linkage to your person takes place.

5. Recipients / processors

The following external providers receive personal data on our behalf. A data processing agreement pursuant to Art. 28 GDPR is in place with each of them.

  • Server hosting: Hetzner Online GmbH, Industriestr. 25, 91710 Gunzenhausen, Germany. Our database, routing engine and map tiles run on a server rented from Hetzner in a German data centre.
  • Geocoding (location search): Komoot GmbH, Berlin, Germany — operates the Photon service (photon.komoot.io). Your search input is forwarded via our server; account data is not transmitted.

6. Retention

  • Account data: until deletion by you. An account can be removed at any time in the app under Settings → Delete account; this irreversibly deletes all associated ratings, locations and sessions.
  • Anonymous sessions: 30 days.
  • Route requests: not persisted.
  • Server log files: a maximum of 14 days for security and stability purposes.
